# 🏆 FIFA World Cup 2026 AI Predictor

An AI-powered tournament simulator that predicts the winner of FIFA World Cup 2026 using Monte Carlo simulation.

## 🧠 How It Works
- Loads 51,653 international match results (1872–2026)
- Calculates team strength using Elo ratings, FIFA rankings & recent form
- Simulates the entire tournament 10,000 times
- Produces win probabilities for all 48 teams

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- XGBoost, Scikit-learn

## 📁 Dataset
- 51,653 international match results (Kaggle)
- Elo ratings for all 48 WC 2026 teams
- FIFA Rankings (June 2026)
